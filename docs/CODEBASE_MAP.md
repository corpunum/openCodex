# Codebase Map

- `src/core/`: agent lifecycle, config, provider plumbing.
- `src/tools/`: command/file/browser tool adapters.
- `src/memory/`: local persistence.
- `src/health/`: health checks and fault isolation.
- `src/ui/server.js`: WebUI API and websocket bridge.
- `tests/`: smoke and WebUI chat e2e checks.
- `deploy/` + `scripts/`: service deployment and installation helpers.

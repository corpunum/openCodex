# Architecture

- `src/core/config.js`: app-home config + provider routing.
- `src/core/agent.js`: autonomous loop and failover.
- `src/core/session-manager.js`: JSON session persistence.
- `src/memory/memory.js`: SQLite memory.
- `src/ui/server.js`: REST + WebSocket server.
- `src/tools/*`: file/shell/git/browser.
